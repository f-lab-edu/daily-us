# 기능 정의
| 카테고리 | 기능명 | 기능설명                                                                                                                                                                                                                                                    |
| --- | --- |---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 회원 | 로그인 | 사용자는 이메일과 비밀번호를 입력하여 로그인할 수 있으며, 소셜 로그인을 통해서도 로그인할 수 있다. 이메일과 비밀번호가 일치하는 경우에만 로그인할 수 있고, 소셜 로그인은 인증 성공 후 가입 여부를 확인하여 로그인 처리한다. 로그인 실패 시 로그인 실패 오류를 반환한다.                                                                                                |
| 회원 | 회원가입 | 사용자는 이메일, 비밀번호, 닉네임, 생년월일을 입력하여 회원가입할 수 있다. 소셜 로그인을 통한 회원가입 시에는 닉네임만 입력한다. 이메일과 닉네임은 중복될 수 없다.                                                                                                                                                          |
| 회원 | 로그아웃 | 사용자는 로그아웃할 수 있으며, 로그아웃 시 세션 또는 토큰을 만료 처리한다.                                                                                                                                                                                                             |
| 회원 | 회원정보 수정 | 사용자는 프로필 이미지, 닉네임, 자기소개를 수정할 수 있다. 수정 완료 시 변경 사항은 프로필 화면에 즉시 반영된다.                                                                                                                                                                                      |
| 회원 | 회원탈퇴 | 사용자는 회원탈퇴를 진행할 수 있다. 작성한 게시글, 댓글, 그룹 참여 이력 처리 방식도 별도 정책으로 정의한다.                                                                                                                                                                                         |
| 회원 | 프로필 조회 | 사용자는 자신의 프로필과 다른 사용자의 프로필을 조회할 수 있다. 프로필 화면에는 프로필 이미지, 닉네임, 자기소개, 게시글 수, 팔로워 수, 팔로잉 수 등의 기본 정보와 활동 요약 정보를 표시한다.                                                                                                                                         |
| 회원 | 프로필 게시글 조회 | 사용자는 프로필의 게시글 탭에서 해당 사용자가 작성한 게시글 목록을 조회할 수 있다. 게시글은 첫 번째 첨부 이미지를 기준으로 표시하며, 최신순으로 정렬하고 삭제된 게시글은 제외한다. 작성일 기준으로 9개씩 페이징 처리를 한다.                                                                                                                         |
| 회원 | 프로필 캘린더 조회 | 사용자는 프로필의 캘린더 탭에서 날짜별 활동 여부를 월 단위 캘린더 형태로 조회할 수 있다. 게시글이 작성된 날짜는 체크 아이콘 또는 별도 표시로 구분하며, 활동 여부는 게시글 작성 여부를 기준으로 판단한다. 미래 날짜에는 활동 기록을 표시하지 않는다.                                                                                                           |
| 회원 | 프로필 그룹 조회 | 사용자는 프로필의 그룹 탭에서 해당 사용자가 속한 그룹 목록을 조회할 수 있다. 그룹명, 그룹 대표 이미지를 표시하며, 사용자가 가입한 그룹만 표시하고 탈퇴한 그룹은 제외한다. 다른 사용자의 프로필에서는 가입된 그룹을 보여주지 않는다. 가입된 그룹은 순서 상관 없이 노출한다.                                                                                              |
| 회원 | 팔로우/언팔로우 | 사용자는 다른 사용자를 팔로우하거나 이미 팔로우한 사용자를 언팔로우할 수 있다. 자기 자신은 팔로우할 수 없으며, 팔로우 또는 언팔로우 시 팔로워 수와 팔로잉 수를 즉시 반영한다.                                                                                                                                                    |
| 게시글 | 게시글 작성 | 사용자는 여러 장의 이미지를 첨부하고 본문 내용을 입력하여 게시글을 작성할 수 있다. 이미지를 첨부하지 않은 게시글은 작성할 수 없으며, 업로드 완료 시 피드 또는 내 프로필 게시글 목록에 반영한다.                                                                                                                                         |
| 게시글 | 게시글 목록 조회(피드) | 사용자는 피드 화면에서 게시글 목록을 조회할 수 있다. 각 게시글에는 작성자 프로필 이미지, 작성자 닉네임, 게시글 이미지, 좋아요 수 또는 좋아요 상태, 댓글 진입 영역, 본문 내용(해시태그 포함)을 표시한다. 피드에는 본인이 팔로우한 사용자 또는 소속된 그룹 멤버의 게시글을 작성 시간 기준으로 정렬하여 표시하며, 팔로우한 사용자와 소속 그룹이 모두 없으면 게시글을 랜덤으로 표시한다. 게시글은 작성일자 기준으로 10개씩 페이징 처리한다. |
| 게시글 | 게시글 상세 조회 | 사용자는 게시글을 선택하여 상세 내용을 조회할 수 있다. 상세 화면에서는 이미지, 본문, 좋아요, 댓글 목록을 함께 확인할 수 있다.                                                                                                                                                                              |
| 게시글 | 게시글 수정 | 작성자는 게시글 우측 상단 메뉴를 통해 게시글을 수정할 수 있다. 게시글 수정 시 이미지와 본문을 변경할 수 있으며, 작성자 본인만 수정할 수 있다.                                                                                                                                                                     |
| 게시글 | 게시글 삭제 | 작성자는 게시글을 삭제할 수 있다. 작성자 본인만 삭제할 수 있으며, 게시글이 삭제되면 해당 게시글의 댓글과 좋아요도 함께 삭제한다.                                                                                                                                                                              |
| 댓글 | 댓글 목록 조회 | 사용자는 게시글 상세 화면에서 댓글 목록을 조회할 수 있다. 각 댓글에는 작성자 프로필 이미지, 작성자 닉네임, 댓글 내용, 작성 시간 또는 순서, 좋아요 수 또는 좋아요 상태를 표시한다.                                                                                                                                               |
| 댓글 | 댓글 작성 | 사용자는 게시글에 댓글을 작성할 수 있다. 로그인한 사용자만 댓글을 작성할 수 있으며, 내용이 없는 댓글은 등록할 수 없다. 댓글이 등록되면 댓글 목록에 즉시 반영한다.                                                                                                                                                          |
| 댓글 | 댓글 좋아요 | 사용자는 댓글에 좋아요를 누르거나 이미 누른 좋아요를 취소할 수 있다. 댓글별 좋아요 수를 반영하며, 사용자당 하나의 댓글에 한 번만 좋아요를 누를 수 있다.                                                                                                                                                                |
| 댓글 | 댓글 삭제 | 댓글 작성자는 본인이 작성한 댓글을 삭제할 수 있다. 작성자 본인만 삭제할 수 있으며, 삭제된 댓글은 목록에서 제거한다.                                                                                                                                                                                     |
| 댓글 | 댓글 수정 | 댓글 작성자는 본인이 작성한 댓글을 수정할 수 있다. 작성자 본인만 수정할 수 있으며, 수정 시 목록에 반영하고 수정 여부를 표시한다.                                                                                                                                                                             |
| 그룹 | 그룹 조회 | 사용자는 그룹 화면에서 특정 그룹의 정보를 조회할 수 있다. 그룹명, 인원 수, 그룹장, 그룹 설명 또는 목적을 표시한다.                                                                                                                                                                                    |
| 그룹 | 그룹 참여 | 사용자는 그룹에 참여할 수 있다. 이미 가입한 그룹에는 중복 참여할 수 없다. 참여 후에는 그룹 랭킹과 구성원 정보를 조회할 수 있다.                                                                                                                                                                             |
| 그룹 | 그룹 탈퇴 | 일반 그룹원은 그룹에서 탈퇴할 수 있다. 탈퇴 시 그룹원 수를 감소시키고, 탈퇴한 회원은 그룹원 목록에서 제외한다.                                                                                                                                                                                        |
| 그룹 | 그룹 해체 | 그룹장은 그룹을 해체할 수 있다. 그룹장만 해체할 수 있으며, 그룹이 해체되면 그룹원, 랭킹, 그룹 데이터를 함께 삭제한다.                                                                                                                                                                                   |
| 그룹 | 그룹 랭킹 조회 | 사용자는 그룹 화면에서 그룹원 랭킹을 조회할 수 있다. 랭킹은 월단위로 순위, 사용자명, 활동 횟수 형태로 표시한다. 활동 횟수는 게시글 업로드 수를 기준으로 한다.                                                                                                                                                            |
| 그룹 | 그룹 목록 조회 | 사용자는 개설되어 있는 그룹의 목록을 조회할 수 있다. 그룹 목록은 순서 상관 없이 10개씩 페이징 처리한다.                                                                                                                                                                                           |
